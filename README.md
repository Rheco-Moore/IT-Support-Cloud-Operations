# IT-Support-Cloud-Operations

This repository demonstrates my hands-on experience in technical support, cloud operations, and troubleshooting across Microsoft 365 and AWS environments.

It includes structured documentation, real-world troubleshooting scenarios, and standard operating procedures (SOPs) that reflect technical issues were investigated, documented, and resolved in a support environment.

## Areas Covered

- Technical Support & Help Desk Operations
- Microsoft 365 Administration (Entra ID, Intune, Exchange Online)
- AWS Environment Support (IAM, EC2, Security Groups)
- Ticket Handling and Incident Documentation
- User Access and Identity Troubleshooting
- SOP Development and Process Documentation

## Purpose

This portfolio is designed to demonstrate:

- Structured troubleshooting approach
- Documentation discipline
- Ability to support both technical and non-technical users
- Experience working within cloud-based environments
- Clear communication and problem-solving methodology

All examples are generalized and do not contain any sensitive or proprietary information.
